## ðŸ“Œ Git Commands Cheat Sheet

| Command                        | Use Case                               |
|--------------------------------|-----------------------------------------|
| `git init`                     | Initialize a new Git repository         |
| `git clone <URL>`              | Clone/download a remote repository      |
| `git add .`                    | Stage all changes                       |
| `git commit -m "message"`      | Commit staged changes with a message    |
| `git push origin <branch>`     | Push local commits to the remote branch |
| `git pull`                     | Fetch and merge latest changes          |
| `git checkout -b <branch>`     | Create and switch to a new branch       |
| `git merge <branch>`           | Merge a branch into current branch      |
| `git status`                   | View modified & staged files            |
| `git log`                      | View commit history                     |
| `git branch`                   | List all branches                       |
| `git branch -d <branch>`       | Delete a branch                         |
| `git reset --hard <commit>`    | Reset working directory to a commit     |
| `git stash`                    | Temporarily save uncommitted changes    |
| `git stash pop`                | Restore stashed changes                 |
